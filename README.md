# Scripts
Some scripts I wrote

## Content
- `vpn`: Shortcuts for the NordVPN CLI on Linux, to have shorter commands
- `reflector-europe`: Bash script that contains all the arguments needed to configure the ArchLinux mirrors in France
- `mpstyle.py`: Python module used to customize `matplotlib`'s plots styles
- `music`: Python script used to batch edit mp3 files' metadata
- `pacman/`: Various scripts for the pacman package manager
